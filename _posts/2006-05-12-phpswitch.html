--- layout: post title: "PHPでのswitch文のトリッキーな使い方" --- こんにちは、ケンジです。

先日の勉強会で初めて知った、switch文のトリッキーな使い方をご紹介します。


if ($a || $b || $c || $d || $e .....) {
    echo "どれかはtrue\n";
}


if文の中の条件がこんな感じでだらだら続くとき。
こんな風に書けます。


switch (true) {
case $a:
case $b:
case $c:
case $d:
case $e:
    echo "どれかは true\n";
}


条件式が複雑に色々と横に並ぶよりも、このように縦に並ぶと読みやすくなりますね。
